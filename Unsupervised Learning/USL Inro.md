## Unsupervised Learning

**Unsupervised Learning** is a type of machine learning where a model is trained using **unlabeled data**.
This means the dataset **does not have correct answers or predefined categories**.

Instead of predicting a known output, the algorithm **analyzes the data by itself** to find **hidden patterns, structures and make groups of similar relationships**.

---

## Core Tasks of Unsupervised Learning

### 1. Clustering

Clustering means **grouping similar data points together**.

**Example:**
In marketing, companies group customers with **similar buying habits** to target them with specific advertisements.

---

### 2. Association Rule Learning

This finds **relationships between items in large datasets**.

**Example:**
In supermarkets, data may show that **customers who buy coffee often also buy milk**.
This is called **Market Basket Analysis**.

---

### 3. Dimensionality Reduction

This technique **reduces the number of features in a dataset** while keeping the most important information.

**Example:**
In **image compression**, unnecessary data is removed to **reduce file size while keeping the image clear**.

---

### 4. Anomaly Detection

This method finds **unusual or rare patterns** in data.

**Example:**
Banks use it to **detect fraud**, such as unusual spending behavior on a credit card.
