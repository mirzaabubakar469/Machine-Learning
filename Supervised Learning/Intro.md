## Supervised Learning

**Supervised Learning** is a type of machine learning where a model is trained using **labeled data**.
This means every **input (feature)** in the dataset already has the **correct output (target)**.

---

## Working

The process can be explained in simple steps:

**1. Training**
The algorithm is given **labeled data**.
Example: Images of cats labeled as **“cat”**.

**2. Pattern Learning**
The model studies the data and finds **patterns** to understand the relationship between **inputs (features)** and **outputs (labels)**.

**3. Improving the Model**
The model makes predictions and compares them with the **correct answers**.
If the prediction is wrong, it **adjusts itself to reduce errors**.

**4. Prediction**
After training, the model can **predict results for new data** that it has never seen before.

---

## Types of Problems

### 1. Classification

Classification means **placing data into categories**. Predecting small number of possible outputs. 

**Example:**
Detecting whether an email is **“Spam” or “Not Spam.”**

---

### 2. Regression

Regression means **predicting a numerical value**. Infinitely many possible outputs.

**Example:**
Predicting the **price of a house** based on its **size, location, and number of rooms**.
